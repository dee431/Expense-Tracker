# Expense-Tracker
A beginner-friendly and interactive Python Expense Tracker that helps you record, view, and calculate your daily spending directly through a simple console menu. This project is perfect for learning Python basics such as loops, lists, dictionaries, and user input handling.

⭐ Features ➕ Add Expense

Add a new expense with:

Date (YYYY-MM-DD)

Category (Food, Transport, etc.)

Description

Amount

All expenses are stored as dictionaries inside a list.

📋 View All Expenses

Displays a clean, numbered list of all recorded expenses.

💵 View Total Spending

Automatically sums all recorded expenses and shows your total kharcha.

🚪 Exit Anytime

Leave the tracker with a friendly message: “Fir Milte!”

🧠 What You Will Learn

This project helps beginners understand:

Python Lists

Dictionaries

While Loops

Conditional Statements

Input Handling

Data Management in Runtime
